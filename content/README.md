# Content Directory

This directory contains MDX content files for articles and lessons.

## Structure

```
content/
  ├── articles/          # Main article content
  │   ├── welcome.mdx
  │   ├── course-overview.mdx
  │   └── ...
  └── examples/          # Code examples and snippets
```

## How to Use

1. Create a `.mdx` file in the appropriate directory
2. Write your content using Markdown syntax
3. Import and use it in your pages

## Example

```mdx
# My Article Title

This is a paragraph with **bold** and _italic_ text.

<CodeBlock code="console.log('Hello')" language="javascript" />
```
